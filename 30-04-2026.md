# 🚀 CI/CD Pipeline Setup using Jenkins on AWS EC2

## 🔹 1. What is CI/CD?

### ✅ Continuous Integration (CI)

* Developers push code frequently to a shared repository (e.g., Git)
* Code is automatically built & tested

### ✅ Continuous Delivery / Deployment (CD)

* Automates:

  * Build
  * Test
  * Release
  * Deployment

### 🔁 Pipeline Flow

```
Plan → Develop → Test → Release → Deploy
```

---

## 🔹 2. CI/CD Tools

Common automation tools:

* Jenkins (most popular, open-source)
* GitHub Actions
* AWS CodePipeline
* Azure DevOps Pipelines

👉 These tools integrate with:

* Git repositories
* Docker
* Kubernetes
* Cloud platforms (AWS, Azure, GCP)

---

## 🔹 3. Architecture Overview

```
Developer → Git → Jenkins Pipeline → Build/Test → Docker → Deploy
```

---

## 🔹 4. Jenkins Server Setup (AWS EC2)

### 📌 Instance Configuration

* OS: Amazon Linux
* Instance Type: t3.large / c7i-flex.large
* Storage: 128 GB SSD
* Security Group:

  * SSH → Port 22
  * Jenkins UI → Port 8080

---

## 🔹 5. Connect to Server

```bash
ssh -i key.pem ec2-user@<public-ip>
```

---

## 🔹 6. System Update

```bash
sudo yum update -y
```

---

## 🔹 7. Set Root Password (Optional)

```bash
sudo passwd
```

---

## 🔹 8. Install Java (Required for Jenkins)

```bash
sudo yum search java-21
sudo yum install java-21-amazon-corretto.x86_64 -y
java --version
```

---

## 🔹 9. Install Git

```bash
sudo yum install git -y

git config --global user.name "Atul Kamble"
git config --global user.email "atul_kamble@hotmail.com"

git config --list
```

---

## 🔹 10. Install Docker

```bash
sudo yum install docker -y

sudo systemctl start docker
sudo systemctl enable docker
```

### 🔹 Fix Docker Permission Issue

```bash
sudo usermod -aG docker $USER
newgrp docker
docker images
```

---

## 🔹 11. Install Jenkins

### ➤ Add Jenkins Repository

```bash
sudo wget -O /etc/yum.repos.d/jenkins.repo \
https://pkg.jenkins.io/rpm-stable/jenkins.repo
```

### ➤ Install Dependencies

```bash
sudo yum upgrade -y
sudo yum install fontconfig java-21-openjdk -y
```

### ➤ Install Jenkins

```bash
sudo yum install jenkins -y
```

---

## 🔹 12. Start Jenkins Service

```bash
sudo systemctl daemon-reload
sudo systemctl start jenkins
sudo systemctl enable jenkins
```

---

## 🔹 13. Add Jenkins to Docker Group

```bash
sudo usermod -aG docker jenkins
sudo systemctl restart jenkins
```

---

## 🔹 14. Access Jenkins UI

Open in browser:

```
http://<EC2-Public-IP>:8080
```

Example:

```
http://98.94.49.170:8080
```

---

## 🔹 15. Unlock Jenkins

Get initial password:

```bash
sudo cat /var/lib/jenkins/secrets/initialAdminPassword
```

Example:

```
9de3d6577b64411b87fa639da0491311
```

## 🔥 Tip 
Most common issues:

* ❌ Docker permission denied → Fix with `usermod + newgrp`
* ❌ Jenkins not opening → Check:

  ```bash
  sudo systemctl status jenkins
  ```
* ❌ Port blocked → Check Security Group (8080 open)
